# Prompt para Stitch: Auto Import Premium

Copia y pega el siguiente texto en Stitch para generar la interfaz inicial:

---

**Act as a World-Class UX/UI Frontend Developer.**
I need you to design and code an ultra-premium digital "Flipbook Catalog" (or interactive horizontal slider) for a high-end B2B/B2C service called **"Auto Import Premium"**. 

**The Business Model:** We act as boutique import brokers. We source premium second-hand cars from Germany (BMW, Audi, Porsche, Mercedes) and deliver them "Turnkey" (Llave en mano) to clients in Luxembourg and Spain. The website acts as our weekly inventory catalog and custom order gateway.

**Aesthetic & Vibe:** 
- **Dark Mode Luxury:** Deep blacks (#0a0a0a), rich dark grays (#1a1a1a), and metallic gold accents (#d4af37) for borders, icons, and call-to-actions.
- **Typography:** 'Playfair Display' for elegant, bold headings. 'Montserrat' or 'Inter' for clean, readable body text.
- **Layout:** It should feel like reading a high-end luxury watch magazine or a Porsche brochure. Use full-screen background images of cars with cinematic lighting, glassmorphism overlays for text boxes, and generous padding.

**Required Sections / "Pages" in the UI:**
1. **Cover Page:** Big bold title "AUTO IMPORT PREMIUM", subtitle "Alemania ➔ Luxemburgo & España". A badge saying "Servicio Llave en Mano".
2. **The Method (How it Works):** A clean, 4-step vertical timeline or grid: 
   - 1. Búsqueda y Verificación (TÜV).
   - 2. Compra Neta Segura (CoC).
   - 3. Logística y Transporte.
   - 4. Matriculación y Entrega.
3. **Featured Cars (The Catalog):** Create a component for a car listing. It should include:
   - A stunning large image of the car.
   - Specs grid (Engine, Mileage, Registration Year, 0-100km/h).
   - A prominent "Precio Llave en Mano" tag.
   - A gold CTA button: "Reservar este Vehículo".
4. **Custom Sourcing (Búsqueda a la Carta):** A sleek, dark-themed contact form. Fields for: Brand, Model, Max Budget, and Destination (Luxembourg or Spain). Submit button: "Solicitar Caza en Alemania".

**Technical Requirements:**
- Use Tailwind CSS for all styling.
- Ensure it is fully responsive (must look incredible on mobile, like a vertical scrollable story or swipeable cards).
- Add subtle micro-animations (e.g., hover effects on car cards that slightly scale the image, pulsing effects on the main CTA).
- Do not use generic placeholders; use Unsplash URLs for luxury cars (e.g., a dark grey Porsche 911 or a black Audi RS6) so the mockup looks production-ready immediately.
